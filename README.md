# Helical and Bevel Gear Design

This project involves designing helical bevel gear using SolidWorks.

## Features
- 3D modeling of helical bevel gear
- Use of CAD tools like sweep, circular pattern, loofted boss/base, revolved boss/base

## Concepts Used
- Gear geometry
- Power transmission
- Mechanical design

## Tools Used
- SolidWorks
